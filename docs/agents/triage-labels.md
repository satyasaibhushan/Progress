# Triage labels

Every triaged implementation issue has one category label and one state label.

| Role | Label | Meaning |
| --- | --- | --- |
| Category | `bug` | Existing behavior is incorrect |
| Category | `enhancement` | New behavior or an improvement |
| State | `needs-triage` | Maintainer evaluation is needed |
| State | `needs-info` | Reporter information is needed |
| State | `ready-for-agent` | Fully specified and ready to implement |
| State | `ready-for-human` | Human implementation or judgment is required |
| State | `wontfix` | The issue will not be actioned |
